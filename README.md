# 🕹️ LinkTree-RetroDark

Um centralizador de links com estética **dark/retro**, desenvolvido para documentar minha evolução no Front-end e praticar conceitos de design e estruturação web.

## 🚀 Sobre o Projeto
Este projeto une a funcionalidade de uma "árvore de links" com uma identidade visual inspirada em interfaces clássicas e sombrias. O foco foi aplicar boas práticas de CSS e semântica HTML, garantindo um layout limpo e responsivo.

### 🛠 Tecnologias Utilizadas
* **HTML5:** Estruturação semântica da página.
* **CSS3:** Flexbox para alinhamento, Box Model e efeitos de transição.
* **Google Fonts:** Utilização da fonte 'VT323' para o visual retro/pixelado.

### 🧠 Conceitos Praticados
* **CSS Reset:** Padronização visual para diferentes navegadores.
* **Estilização de Listas:** Remoção de marcadores padrão para criação de botões personalizados.
* **Hover Effects:** Feedback visual interativo nos links.
* **Responsividade:** Ajustes para visualização em dispositivos móveis.
